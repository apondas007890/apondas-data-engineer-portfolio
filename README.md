## About This Portfolio

I build reliable data pipelines, clean analytical datasets, and scalable data systems that turn raw information into trusted insight.

This portfolio has two major experiences:

### 1. Visual Portfolio

A polished dark portfolio interface with animations, project cards, technical skills, experience, education, certifications, practice dashboard, resume, and contact page.

### 2. SQL Shades / SSMS Inspired Portfolio

A SQL Server Management Studio inspired interface where visitors can explore my portfolio like a database using SQL-style commands and view-based navigation.
